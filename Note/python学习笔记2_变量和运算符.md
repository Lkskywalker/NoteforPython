### 变量

##### 定义变量

Python是强类型 语言

赋值不需要指定数据类型,赋值是什么类型,变量就是什么类型

```python
a = 10
b = 'Hello'
```

##### 变量的数据类型

![](https://s1.ax1x.com/2022/10/03/xMjbEq.png)

用<mark>type()</mark>函数查看数据类型

```python
a=10
print(type(a))
b=1.1
print(type(b))
c='Tom'
print(type(c))
d=True
print(type(d))
#元组
e=()
print(type(e))
#列表
f=[]
print(type(f))
#字典
g={}
print(type(g))
```

##### 变量的命名规则

1. 必须以字母或下划线开头

2. 变量区分大小写
